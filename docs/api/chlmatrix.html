Creates a new matrix from allocated memory.

Creates a new matrix from allocated memory. This function returns NULL if creation fails. The returned matrix must be freed with chl_matrix_free() when no longer in use. chl_matrix_free() will not free the memory allocated for entries.

  • n_columns – number of columns

int chl_matrix_add(ChlMatrix a, ChlMatrix b, ChlMatrix *c_ptr)¶

Adds two matrices.

Adds two matrices of equal shape and sets the result to the matrix c. The dimensions of a and b must be equal.

c may be NULL. If c is NULL, a new matrix will be created. This newly created matrix must be freed with chl_matrix_free() when no longer in use.

If c is not NULL, c must have the same dimensions of a and b.

Parameters

  • a – a matrix

  • b – another matrix

  • c_ptr – pointer to sum of a and b

Returns

0 if operation was a success, -1 otherwise

int chl_matrix_mult(ChlMatrix a, ChlMatrix b, ChlMatrix *c_ptr)¶

Multiplies two matrices.

Multiplies two matrices a and b and sets the result to the matrix c. The inner dimensions of a and b must be equal.

c may be NULL. If c is NULL, a new matrix will be created. This newly created matrix must be freed with chl_matrix_free() when no longer in use.

If c is not NULL, c must have dimensions equal to the outer dimensions of a and b.

Parameters

  • a – left matrix

  • b – right matrix

  • c_ptr – pointer to product of a and b

Returns

0 if operation was a success, -1 otherwise
